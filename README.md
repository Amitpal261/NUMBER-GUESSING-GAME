# 🎮 Number Guessing Game

A simple and interactive **Number Guessing Game** built using **HTML, CSS, and JavaScript**.

In this game, the system generates a random number and the player must guess the correct number. After each guess, the game provides feedback indicating whether the guess is **too high, too low, or correct**.

This project demonstrates **JavaScript logic, DOM manipulation, and basic game development in the browser**.

---

## 🚀 Features

- Random number generation
- User input validation
- Hint system (Too High / Too Low)
- Correct guess notification
- Attempt tracking
- Reset / Play again functionality
- Simple and responsive UI

---

## 🛠️ Tech Stack

**Frontend**

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🎯 How the Game Works

1. The game generates a **random number** within a defined range.
2. The player enters a guess in the input field.
3. The system checks the guess and provides feedback:
   - 📈 Too High
   - 📉 Too Low
   - 🎉 Correct Guess
4. The game tracks the number of attempts.
5. The player can reset the game and play again.
